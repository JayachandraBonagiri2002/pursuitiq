PursuitIQ Knowledge Base — Proposal Documents
===============================================

Place your winning proposal documents here (PDF, DOCX, PPTX).

Then upload them via:
  1. The web UI at http://localhost:3000/knowledge
  2. The API: POST /api/knowledge/upload

Each document you add makes the AI smarter:
  - Agent 2 learns your win patterns
  - Agent 5 learns your pricing patterns
  - Agent 6 learns your writing style

Recommended metadata when uploading:
  - industry: Banking, Insurance, Telecom, etc.
  - client_name: The client this proposal was for
  - geography: Germany, UK, USA, etc.
  - deal_size: $50M, EUR 30M, etc.
  - outcome: WON or LOST
  - tags: cloud, migration, SAP, etc.
